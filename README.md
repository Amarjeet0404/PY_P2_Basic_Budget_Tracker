## 📌 Project Overview

The **Budget Tracker** is a simple Python-based command-line application that helps users **manage their personal finances** by tracking income and expenses.

Users can **add their salary, record expenses, and monitor the remaining budget**. The program stores all expenses and displays them along with the remaining balance.

This project demonstrates the use of:

* Object-Oriented Programming (OOP)
* Lists and dictionaries
* User input handling
* Loops and conditional statements

---

## 🎯 Objective

The goal of this project is to build a **simple financial management tool** that allows users to:

* Track their expenses
* Monitor remaining budget
* Maintain a list of spending activities
* Improve personal budgeting habits

---

## ✨ Features

* 💵 Add salary to the budget
* 🧾 Record multiple expenses
* 📊 View all expenses with descriptions
* 💰 Display remaining budget
* 🔁 Continue tracking expenses after salary update
* 🚪 Exit option when tracking is complete

## ⚙️ How the Program Works

1. User enters their **salary amount**.
2. The program allows the user to **add multiple expenses**.
3. Each expense includes:

   * Description
   * Amount
4. The expense amount is **deducted from the budget**.
5. All expenses are stored and displayed.
6. The remaining budget is calculated and shown.
